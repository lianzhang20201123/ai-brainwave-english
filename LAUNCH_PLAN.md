### 部署步骤（10分钟）

#### 第1步：准备代码（2分钟）
```bash
# 1. 初始化 Git 仓库
git init

# 2. 添加所有文件
git add .

# 3. 提交代码
git commit -m "Initial commit: AI脑波英语网站"

# 4. 推送到 GitHub
git branch -M main
git remote add origin https://github.com/你的用户名/ai-brainwave-english.git
git push -u origin main
```

#### 第2步：创建 Vercel 项目（3分钟）
1. 访问 [Vercel](https://vercel.com/)
2. 点击 "Sign Up" 或 "Login"
3. 使用 GitHub 账号登录
4. 点击 "Add New" → "Project"
5. 选择你的 GitHub 仓库
6. 点击 "Import"

#### 第3步：配置部署（2分钟）
1. **Framework Preset：** 选择 "Other" 或 "Static"
2. **Root Directory：** 留空（默认）
3. **Build Command：** 留空（无需构建）
4. **Output Directory：** 留空（默认）
5. 点击 "Deploy"

#### 第4步：等待部署（2分钟）
- Vercel 会自动部署你的网站
- 部署完成后，你会得到一个 URL，如：`https://ai-brainwave-english.vercel.app`

#### 第5步：测试（1分钟）
1. 访问生成的 URL
2. 测试所有页面
3. 测试移动端显示
4. 确认一切正常

#### 第6步：（可选）配置自定义域名（如需要）
1. 购买域名（如 ai-naobo.com）
2. 在 Vercel 控制台添加域名
3. 配置 DNS 解析（通常需要添加 CNAME 记录）
4. 等待 DNS 生效（最多 48 小时，通常 10 分钟即可）

## 方案C：阿里云OSS + CDN（国内方案 ⭐⭐⭐⭐）

### 优势
- ✅ **国内访问速度快** - 国内CDN加速
- ✅ **稳定可靠** - 阿里云大厂背书
- ✅ **安全可信** - 国内服务器，数据更安全
- ✅ **支持备案** - 合规运营

### 劣势
- ❌ **需要备案** - 需要企业资质，审核时间长
- ❌ **有费用** - 需要支付存储和流量费用
- ❌ **配置复杂** - 需要配置OSS、CDN、域名解析
- ❌ **无法立即上线** - 备案需要 15-20 个工作日

### 费用明细（预估）
- **OSS存储：** 约 20 元/年（假设存储 1GB）
- **CDN流量：** 约 30-50 元/年（假设每月 10GB 流量）
- **域名费用：** 约 60-80 元/年
- **总计：** **约 110-150 元/年**

### 适用场景
- ✅ 必须备案
- ✅ 追求国内访问速度
- ✅ 有企业资质
- ✅ 有一定的预算

### 备案流程
1. **购买阿里云服务器**（虚拟机或轻量应用服务器）
2. **购买域名**（在阿里云或其他注册商）
3. **提交备案申请**
   - 提供营业执照
   - 提供法人身份证
   - 提供网站负责人身份证
4. **等待审核**（15-20 个工作日）
5. **备案成功后**才能部署网站

### 部署步骤
1. **购买 OSS 存储**
2. **上传网站文件**
3. **配置 CDN 加速**
4. **配置自定义域名**
5. **配置 HTTPS 证书**

### 官方资源
- **官方网站：** https://www.aliyun.com/product/oss
- **文档：** https://help.aliyun.com/product/31815.html
