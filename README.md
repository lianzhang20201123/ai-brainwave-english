# AI脑波英语官网静态站

适用于 GitHub + Vercel 部署。

## 本地结构

- `index.html` 首页
- `principle.html` 核心原理
- `courses.html` 课程体系
- `cases.html` 成功案例
- `safety.html` 安全认证
- `partner.html` 机构合作
- `contact.html` 联系我们
- `assets/` 图片、视频、文档、样式、脚本

## 部署到 GitHub

1. 新建 GitHub 仓库
2. 上传全部文件到仓库根目录
3. 确认 `assets/` 文件夹完整上传

## 部署到 Vercel

1. 登录 Vercel
2. 导入 GitHub 仓库
3. Framework Preset 选择 `Other`
4. Build Command 留空
5. Output Directory 留空
6. 点击 Deploy
7. 在 Vercel 项目设置中绑定域名 `ai-naobo.com`

## 域名与 SEO

- sitemap: `https://ai-naobo.com/sitemap.xml`
- robots: `https://ai-naobo.com/robots.txt`
- 所有页面均已写入基础 meta 信息和 canonical
